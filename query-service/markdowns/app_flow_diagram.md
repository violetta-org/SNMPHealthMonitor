## 4. Real-time Data Flow (UDP → WebSocket)

```
┌─────────────────────────────────────────────────────────┐
│              External Service                            │
│         (Sends UDP notification)                         │
└────────────────────┬────────────────────────────────────┘
                     │
                     ▼
┌─────────────────────────────────────────────────────────┐
│         UDPNotificationListener._listen()                │
│  ┌──────────────────────────────────────────────────┐   │
│  │ • Receive UDP message                            │   │
│  │ • Parse JSON                                     │   │
│  │ • Validate event = "new_data"                    │   │
│  └──────────────────────────────────────────────────┘   │
└────────────────────┬────────────────────────────────────┘
                     │
                     ▼
┌─────────────────────────────────────────────────────────┐
│              on_new_data() Callback                      │
│  ┌──────────────────────────────────────────────────┐   │
│  │ • Extract: sysname, metric_count, timestamp      │   │
│  │ • ws_manager.get_active_topics(sysname)          │   │
│  └──────────────────────────────────────────────────┘   │
└────────────────────┬────────────────────────────────────┘
                     │
                     ▼
┌─────────────────────────────────────────────────────────┐
│         For each active topic:                           │
│  ┌──────────────────────────────────────────────────┐   │
│  │ asyncio.run_coroutine_threadsafe(                │   │
│  │   stream_topic_data(sysname, topic, timestamp)   │   │
│  │ )                                                │   │
│  └──────────────────────────────────────────────────┘   │
└────────────────────┬────────────────────────────────────┘
                     │
                     ▼
┌─────────────────────────────────────────────────────────┐
│         stream_topic_data()                              │
│  ┌──────────────────────────────────────────────────┐   │
│  │ • get_topic_data(sysname, topic, timestamp)      │   │
│  │ • ws_manager.stream_data(sysname, topic, data)   │   │
│  └──────────────────────────────────────────────────┘   │
└────────────────────┬────────────────────────────────────┘
                     │
         ┌───────────┴───────────┐
         │                       │
         ▼                       ▼
┌──────────────────┐   ┌──────────────────────────────────┐
│ Database Query   │   │ ws_manager.stream_data()         │
│ (see section 5)  │   │ • Get all websockets for topic   │
└────────┬─────────┘   │ • Send JSON to each websocket    │
         │             │ • Cleanup disconnected            │
         │             └──────────┬───────────────────────┘
         │                        │
         └────────────┬───────────┘
                      │
                      ▼
         ┌────────────────────────────┐
         │ Clients Receive Real-time  │
         │ Data via WebSocket         │
         └────────────────────────────┘
```

## 5. Database Query Flow

```
┌─────────────────────────────────────────────────────────┐
│              Query Function Called                       │
│  (get_system_metrics, get_cpu_metrics, etc.)            │
└────────────────────┬────────────────────────────────────┘
                     │
                     ▼
┌─────────────────────────────────────────────────────────┐
│         Calculate Time Cutoff                            │
│  ┌──────────────────────────────────────────────────┐   │
│  │ if notify_timestamp:                             │   │
│  │   cutoff = timestamp - 30 seconds                │   │
│  │ else:                                            │   │
│  │   cutoff = now() or now() - 1 minute             │   │
│  └──────────────────────────────────────────────────┘   │
└────────────────────┬────────────────────────────────────┘
                     │
                     ▼
┌─────────────────────────────────────────────────────────┐
│         get_db() Context Manager                         │
│  ┌──────────────────────────────────────────────────┐   │
│  │ • create_connection()                            │   │
│  │ • Yield connection                               │   │
│  └──────────────────────────────────────────────────┘   │
└────────────────────┬────────────────────────────────────┘
                     │
                     ▼
┌─────────────────────────────────────────────────────────┐
│         Execute SQL Query                                │
│  ┌──────────────────────────────────────────────────┐   │
│  │ • Query with time filter                         │   │
│  │ • Use window functions (ROW_NUMBER) if needed    │   │
│  │ • Calculate rates (for network/disk I/O)         │   │
│  │ • Apply pagination (for diskio)                  │   │
│  └──────────────────────────────────────────────────┘   │
└────────────────────┬────────────────────────────────────┘
                     │
                     ▼
┌─────────────────────────────────────────────────────────┐
│         Serialize Results                                │
│  ┌──────────────────────────────────────────────────┐   │
│  │ serialize_row() or serialize_rows()              │   │
│  │ • Convert datetime → ISO format strings          │   │
│  └──────────────────────────────────────────────────┘   │
└────────────────────┬────────────────────────────────────┘
                     │
                     ▼
┌─────────────────────────────────────────────────────────┐
│         Return Dictionary                                │
│  {device_info: {...}, topic_data: {...}}                │
└─────────────────────────────────────────────────────────┘
```

## 9. Shutdown Flow

```
┌─────────────────────────────────────────────────────────┐
│              Application Shutdown Signal                 │
│              (Ctrl+C or system signal)                   │
└────────────────────┬────────────────────────────────────┘
                     │
                     ▼
┌─────────────────────────────────────────────────────────┐
│         lifespan() - Shutdown Phase                      │
│  ┌──────────────────────────────────────────────────┐   │
│  │ 1. notify_listener.stop()                        │   │
│  │    • Set running = False                         │   │
│  │    • Close UDP socket                            │   │
│  │    • Wait for thread to finish                   │   │
│  │ 2. Cleanup resources                             │   │
│  └──────────────────────────────────────────────────┘   │
└────────────────────┬────────────────────────────────────┘
                     │
                     ▼
┌─────────────────────────────────────────────────────────┐
│              Uvicorn Graceful Shutdown                   │
│         (timeout_graceful_shutdown: 10s)                 │
└─────────────────────────────────────────────────────────┘
```
